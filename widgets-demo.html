GradeEarn Widget Renderer — §110 Demo

Each panel renders one widget type from a tiny JSON spec. STAAR-faithful conventions (Verdana, navy fills, thin black strokes, stacked fractions on number-line labels). All renderers vanilla JS + inline SVG; no React, no bundler.

fraction-bar

The IXL-screenshot widget. parts = denominator, filled = numerator. Cell widths are integer pixels (sacred equal-parts rule).

number-line

Major + minor ticks, arrows both ends, fraction labels rendered as stacked num/den (STAAR convention). Optional marker points + inequalities.

plotter

Bar / dot / line / histogram. Single uniform fill color across all bars (STAAR convention).

table

HTML table (semantic + a11y-friendly), Verdana font, bold header row.

area-model

Two flavors: multi-digit multiplication (place-value rectangles + partial products) and fraction × fraction (overlap-shaded grid).

tape-diagram (Tier 2)

STAAR grade 3-5 strip diagram. Black outline + labeled parts; optional total brace.

base-10-blocks (Tier 2)

STAAR K-3 place-value. Flat 2D (not isometric) per TEA convention. Black-on-white outline.

shape-2d (Tier 2)

Labeled rectangles, triangles, circles, right triangles, polygons. Outline-only by default per STAAR.

clock-face (Tier 2)

Analog clock with 12 hour numbers + 60 minute ticks. STAAR grades 1-3.

Error handling

Malformed specs render a tiny "Diagram unavailable" placeholder — never throw, never blank.
