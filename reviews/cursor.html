However, the Agent feature (the really powerful stuff) uses a separate usage pool that can deplete quickly on complex tasks. Heavy agent users should budget for Pro+ ($60/mo) or Ultra ($200/mo). Also, "Cloud Agents" means your code is sent to their servers for processing‚ÄîPrivacy Mode exists but limits functionality.
