💡 What the marketing page doesn't tell you

1. Free tier isn't bait-and-switch. Unlike competitors, Codeium's free tier is genuinely unlimited. No credit limits, no time caps. The catch: your code is used to improve models (opt-out available).

2. Autocomplete beats chat (for now). Inline suggestions are excellent. Chat feature is solid but not Cursor-level. Use it for completions first, chat second.

3. Enterprise pricing is opaque. "Let's talk" means negotiation. Expect $30-60/seat/mo based on team size. No self-serve pricing transparency.

Real-World Use Cases We Tested

👨‍💻 Full-Stack Developer (Personal Use)

What we used it for: React/TypeScript frontend, Python backend, SQL queries, documentation writing.

What worked: Autocomplete was 80-90% accurate. Function signature completion saved hours. Multi-line suggestions often exactly what I needed. Zero lag in VS Code.

What didn't: Chat sometimes gave outdated library syntax. No codebase-wide context (unlike Cursor). Refactoring suggestions basic.

🏢 Engineering Team (10 developers)

What we used it for: Migrated from GitHub Copilot to save $10/user/mo ($100/mo total).

What worked: Admin dashboard shows usage stats. Team onboarding trivial (browser extension install). SOC 2 compliance satisfied security team.

What didn't: No usage quotas per user (can't prevent abuse). Analytics limited vs paid tools. Support response slow on free tier.

What the Privacy Policy Actually Says

Free tier: Your code may be used to improve models unless you opt-out. Telemetry collected (completions accepted/rejected). Full policy →

Data collection: Code snippets (context for autocomplete), usage telemetry, crash reports. NOT full files or repos (only active file context).

Training usage (Free tier): May use anonymized code snippets to improve models. Can opt-out in settings (won't affect quality).

Teams/Enterprise: Your code is NOT used for training. Zero-retention policy (processed, not stored).

Key privacy wins: SOC 2 Type II certified. No code stored long-term (even on free). Opt-out respects choice immediately.
