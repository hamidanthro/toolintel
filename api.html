Core Endpoints

GET /tools

Returns a paginated list of all reviewed tools with their overall score, category, subcategory, review date, and methodology version.

Query Parameters

ParameterTypeDescription
categorystringFilter by category (e.g., "Foundation Models")
minScoreintegerMinimum overall score (0-100)
compliancestringRequired compliance (e.g., "hipaa,soc2")
regionstringData residency region
pricingTierstringFilter by pricing tier
pageintegerPage number (default: 1)
perPageintegerResults per page (max: 50)

Example Request

curl -H "X-API-Key: YOUR_API_KEY" \ "https://api.toolintel.ai/tools?category=Foundation%20Models&minScore=80"

Response

{
  "tools": [
    {
      "slug": "claude",
      "name": "Claude",

      "subcategory": "LLM",

      "reviewDate": "2026-02-20",
      "methodologyVersion": "1.0"

  ],
  "total": 45,
  "page": 1,
  "perPage": 50
}

Browser Extension Specification

Build a browser extension that displays ToolIntel scores on AI tool websites. This specification is open source and available on GitHub.

Sample Chrome Extension Code

// content.js - Detect domain and show score overlay const domain = window.location.hostname.replace('www.', ''); // Map domains to tool slugs const domainMap = { 'claude.ai': 'claude', 'chat.openai.com': 'gpt-4', 'gemini.google.com': 'gemini' }; const toolSlug = domainMap[domain]; if (toolSlug) { fetch(`https://api.toolintel.ai/tools/${toolSlug}`, { headers: { 'X-API-Key': API_KEY } }) .then(r => r.json()) .then(data => showScoreOverlay(data)); } function showScoreOverlay(tool) { const overlay = document.createElement('div'); overlay.innerHTML = ` <div class="toolintel-badge"> <strong>ToolIntel: ${tool.overallScore}/100</strong> <small>Privacy: ${tool.categoryScores.dataPrivacySecurity.score}</small> <small>Compliance: ${tool.categoryScores.compliance.score}</small> <a href="https://toolintel.ai/reviews/${tool.slug}">Full Review ‚Üí</a> </div> `; document.body.appendChild(overlay); }
