Sebastian is collecting Bluebonnets for a school project. He picked 25 Bluebonnets on Monday and 32 Bluebonnets on Tuesday. How many Bluebonnets did Sebastian pick in total?

  1. 57 ✓
  2. 62
  3. 50
  4. 45

Why: To find the total number of Bluebonnets Sebastian picked, we need to add the number of Bluebonnets he picked on Monday (25) to the number he picked on Tuesday (32). So, 25 + 32 = 57. Therefore, Sebastian picked a total of 57 Bluebonnets.

  3. 207
  4. 215

Why: To find the total number of seashells Avery collected, you need to add the number of seashells she collected on Saturday and Sunday. So, 125 + 87 = 212. Therefore, Avery collected 212 seashells in total.

  1. 557 ✓

  4. 537

Why: To find the total number of oranges sold over the weekend, we can add the number of oranges sold on Saturday (245) to the number sold on Sunday (312). So, 245 + 312 = 557. Therefore, the correct answer is 557.

  1. 167 ✓
  2. 178
  3. 145
  4. 156

Why: To find out how many stickers Levi has left after giving some away, you need to subtract the number of stickers he gave to his friend from the total number of stickers he originally had. So, 245 - 78 = 167. Therefore, Levi has 167 stickers left.
